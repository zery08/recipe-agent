# Recipe Compare Skill

Compare recipes by following this sequence:

1. Normalize recipe identifiers, names, and aliases.
2. Define comparable metrics before querying.
3. Use matching time windows and comparable cohorts.
4. Separate totals, rates, normalized values, and cost metrics.
5. Call out missing data, small samples, and biased comparisons.
6. End with a concise conclusion and the evidence behind it.
